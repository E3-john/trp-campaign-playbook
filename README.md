# TRP Campaign Playbook Builder Interface

Co-op Enrollment & Submission portal for Truckworx dealerships.

## Run Locally

1. Clone this repo:
2.    ```
         git clone https://github.com/E3-john/trp-campaign-playbook.git
         ```
      2. Open `index.html` in your browser
  
      3. No build step needed - it's a single HTML file using Tailwind CSS via CDN.
  
      4. ## Features
  
      5. - Campaign selection sidebar (Spring Sales Event, Target Account Sales Blitz, Inventory Clearance, Custom Co-op Request)
         - - Interactive playbook with campaign details, timeline, marketing assets, budget breakdown, and approval workflow
           - - Expandable/collapsible campaign timeline sections
             - - Budget progress bars
               - - Responsive layout with Tailwind CSS
